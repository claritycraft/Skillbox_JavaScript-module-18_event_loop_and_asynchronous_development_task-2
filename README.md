### En:
#### Task 2
What You Need to Do
- Create a div element with id="progress-bar", a height of 20px, and a maximum width of 500px.
Style it however you like — the main thing is that it’s visible on the page.
- Create a function called progress(time) where time is the number of seconds.
This function should animate the progress bar so that it starts with a width of 0px and gradually expands to the maximum width over the specified time.
It’s recommended to use transform: scaleX() for the animation.
The minimum duration for the animation should be 2 seconds.
The animation must be smooth.
- Optional:
Below the progress bar, display a block showing the number of whole seconds passed, updating once per second.

Acceptance Criteria
- The animation and timer update only during the specified time — the timer stops when the bar finishes expanding, and vice versa.
- The progress bar expands smoothly over the given duration.

### Ru:
#### Задача 2
Создание progress bar
Что нужно сделать
Создайте элемент div с id=”progress-bar”, высотой 20 px и максимальной шириной 500 px. Стилизуйте его на своё усмотрение — главное, чтобы элемент был виден на странице.
Создайте функцию progress(time) (time — количество секунд), которая анимирует блок progress bar так, чтобы сначала он имел ширину 0 pх, а затем за заданное время растягивался на максимальную ширину (советуем использовать transform: scaleX()). Минимальное время, за которое растягивается блок, — две секунды. Анимация должна быть плавной.
По желанию. Под блоком progress bar настройте вывод блока с количеством прошедших целых секунд, обновляющийся раз в секунду.

Критерии приёма задания
Анимация и таймер изменяются только в течение заданного времени, то есть таймер не меняется после завершения растягивания блока и наоборот.
Блок плавно растягивается в течение заданного времени.
